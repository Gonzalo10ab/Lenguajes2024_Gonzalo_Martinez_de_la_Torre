Validación XML Schema

XML Schema (XSD) permite definir la estructura y restricciones de un documento XML

Características de XML

  • Es un archivo XML
  • Usa espacio de nombre
  • Su extensión es .XSD
  • Usa espacio de nombre

Utilidad

    El propósito del esquema XML es definir

  • Los elementos y atributos que pueden aparecer en un documento
  • El número y orden elementos secundarios
  • Los tipos de datos para elementos y atributos
  • Los valores predeterminados y fijos para elementos y atributos

Tipos de datos

Tipo Significado
string Secuencia de caracteres.
boolean Toma los valores true o false.
decimal Número decimal con precisión arbitraria.
Integer Número entero positivo o negativo.
positiveInteger Número entero positivo.
negativeInteger Número entero negativo.
float Número de punto flotante de 32 bits de precisión simple.
double Número de punto flotante de 64 bits de precisión doble.
dateTime Hora y fecha en formato “YYYYMM-DDThh:mm:ss”
date Fecha en formato “YYYY-MM-DD”
time Hora en formato “hh:mm:ss”

Elementos Simples

Un elemento simple es aquel que contiene solo texto y no tiene atributos ni elementos secundarios. Se definen en XML Schema utilizando el tipo de dato correspondiente.

<xs:element name="nombre" type="xs:string"/>

Elementos Complejos

Los elementos complejos pueden contener otros elementos o atributos. Se definen utilizando <xs:complexType>. 

<xs:element name="persona">
    <xs:complexType>
        <xs:sequence>
            <xs:element name="nombre" type="xs:string"/>
            <xs:element name="edad" type="xs:integer"/>
        </xs:sequence>
    </xs:complexType>
</xs:element>

Restricciones

Se pueden aplicar restricciones a los valores de los elementos utilizando <xs:restriction>. Ejemplo de restricción en un número entero

Restricción Significado
minInclusive Mínimo valor del número.
minExclusive El número debe ser mayor que el indicado aquí.
maxInclusive Máximo valor del número.
maxExclusive El número debe ser menor que el indicado aquí.
totalDigits Número total de dígitos en el número.
fractionDigits Número de cifras decimales.
length Nº de caracteres del literal o elementos en lista.
minLength Nº de caracteres mínimo literal o en lista.
maxLength Número de caracteres máximo del literal o en lista.
pattern Patrón que se tiene que cumplir indicado con expresiones regulares. 

<xs:element name="edad">
    <xs:simpleType>
        <xs:restriction base="xs:integer">
            <xs:minInclusive value="0"/>
            <xs:maxInclusive value="120"/>
        </xs:restriction>
    </xs:simpleType>
</xs:element>
