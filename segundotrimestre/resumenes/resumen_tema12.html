Resumen de XPath y XML

XPath es un lenguaje que permite la navegación y selección de nodos en documentos XML.

¿Qué es XPath?

XPath es un lenguaje de consulta para seleccionar nodos dentro de documentos XML basado en una estructura jerárquica.

Se usa ampliamente en combinación con tecnologías como XSLT y XQuery para manipular y transformar documentos XML.

Tipos de nodos en XPath

XPath trabaja con diferentes tipos de nodos en un documento XML:

  • Raíz: Representa el nodo raíz del documento, indicado con /.
  • Elemento: Cualquier etiqueta XML en el documento.
  • Atributo: Información asociada a un elemento, representada con @nombre_atributo.
  • Texto: Contenido dentro de un elemento.

Expresiones XPath

Algunas expresiones XPath comunes para seleccionar nodos:

  • /concesionario/coche - Selecciona todos los coches dentro del concesionario.
  • //coche - Selecciona todos los coches en el documento, sin importar su ubicación.
  • //coche[@matricula="8280KWT"] - Encuentra el coche con la matrícula 8280KWT.
  • //coche[2] - Selecciona el segundo coche del documento.
  • //coche[precio>20000]/marca - Obtiene la marca de los coches con precio superior a 20,000.

Funciones en XPath

XPath permite realizar funciones sobre nodos y atributos:

  • count(//coche) - Cuenta el número de coches en el documento.
  • contains(text(), "Azul") - Verifica si un nodo de texto contiene la palabra "Azul".
  • starts-with(@matricula, "82") - Encuentra los coches cuya matrícula comienza con "82".
  • last() - Selecciona el último elemento dentro de un conjunto de nodos.

Ejemplo XML con XPath

<concesionario>
    <coche matricula="6732KPC">
        <marca>Seat Ateca</marca>
        <combustible>Gasolina</combustible>
        <precio>23000</precio>
    </coche>
    <coche matricula="8280KWT">
        <marca>Volkswagen Golf</marca>
        <combustible>Diesel</combustible>
        <precio>25000</precio>
    </coche>
</concesionario>

Consultas XPath sobre este XML:

  • //coche[@matricula="6732KPC"]/marca - Devuelve "Seat Ateca".
  • //coche[combustible="Gasolina"] - Selecciona coches con combustible gasolina.
  • count(//coche) - Cuenta el número total de coches.
