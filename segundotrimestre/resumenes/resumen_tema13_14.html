Resumen de Transformaciones XSLT

Es un estandar de la organización W3C que presenta una forma de transformar documentos XML

¿Qué es XSLT?

XSLT (eXtensible Stylesheet Language Transformations) es un lenguaje basado en XML que permite transformar documentos XML en otros formatos como HTML, XML o texto plano.

Las transformaciones pueden ser visuales y estructurales, facilitando la conversión de un documento XML en otro tipo de documento mediante el uso de patrones y reglas.

Elementos Clave de XSLT

  • <xsl:stylesheet>: Define el inicio de un documento XSLT.
  • <xsl:template>: Contiene reglas de transformación.
  • <xsl:apply-templates>: Aplica plantillas a elementos coincidentes.
  • <xsl:value-of>: Extrae el contenido de un nodo.
  • <xsl:for-each>: Itera sobre un conjunto de nodos.
  • <xsl:choose>: Define condiciones mediante <xsl:when> y <xsl:otherwise>.
  • <xsl:output>: Indica el formato del documento de salida (XML, HTML, texto).
  • <xsl:sort>: Ordena los resultados dentro de una estructura iterativa.

Proceso de Transformación en XSLT

El procesador XSLT analiza el documento XML y construye un árbol de documentos. Luego, recorre el árbol desde la raíz y aplica las plantillas definidas en el XSLT. Si se encuentra una coincidencia con un nodo, se aplica la transformación correspondiente.

Ejemplo de Transformación XSLT

Este es un ejemplo básico de cómo transformar un XML en HTML utilizando XSLT:

<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">
    <xsl:template match="/">
        <html>
            <body>
                <h1>Listado de Pokémon</h1>
                <xsl:for-each select="pokeDAW/pokemon">
                    <p><xsl:value-of select="nombre"/> - Tipo: <xsl:value-of select="@tipo"/></p>
                </xsl:for-each>
            </body>
        </html>
    </xsl:template>
</xsl:stylesheet>

Aplicaciones de XSLT

Algunas aplicaciones comunes de XSLT incluyen:

  • Transformación de datos XML en HTML para su visualización web.
  • Generación de informes a partir de bases de datos XML.
  • Integración de sistemas mediante conversión de formatos XML.
  • Formato avanzado de documentos XML mediante XSL-FO para la generación de archivos PDF, SVG o RTF.

Herramientas para Trabajar con XSLT

Existen varias herramientas para desarrollar y probar transformaciones XSLT:

  • Oxygen XML Editor: Editor avanzado para trabajar con XML y XSLT.
  • Altova XMLSpy: Herramienta de desarrollo XML con soporte para XSLT.
  • Saxon: Procesador XSLT open-source.
  • Xalan: Procesador XSLT de código abierto.
